# About testing
```
	Unit testing:
		UT (Unit test):
		UT is a piece of code (typically a method) written by developer that execute specific functionality in code
		and assert a behavior
		
		Test coverage:
		The percentage % of code which is tested by UT is called test coverage

		Unit testing:
		To execute UT

	Integration testing (Functional testing):
		To test the system as whole
		
	Performance testing:
		code under test run is fast enough even if it is under high load
```

## Unit testing framework:
Junit, TestNG
